# Weatherly

Weatherly is a Flutter weather app planned around a premium dark
glassmorphism UI, Open-Meteo weather data, Clean Architecture, and a
feature set focused on current weather, rain timing, hourly forecast,
weekly forecast, air quality, map-based location selection, and useful
daily recommendations.
